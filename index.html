FAQ

Fast answers to common questions.

What files do you need?

Vector files are best (AI/EPS/SVG). High-res PNG works too. If you’re not sure, send what you have — we’ll guide you.

Can you print front + back?

Yes. Multiple locations are available (front, back, sleeve, etc.). Extra print location is +$3/shirt.

Do you offer bulk pricing?

Yes — pricing improves as quantity increases. If you’re doing 100+ or repeat orders, ask about deeper discounts.
