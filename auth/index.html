One portal for admins now, customers next.

The auth surface is shared on purpose. Admin review tools are first, and the same portal can expand into customer order history, approvals, and payment preferences without moving off the Gun/Vercel setup.

  • Admin review tools live first SEA authenticates the user and the current dashboard admits admins by pub allowlist or the local third-eye-print-co/admins graph.
  • Customer workspace next Past orders, approvals, and saved billing details can be layered onto the same portal once the order graph is defined.
  • Small serverless surface Vercel only handles public runtime config and Stripe checkout session creation.

Right now only the existing tmsteph@3dvr admin identity bootstraps with its matching SEA public key. Any additional Third Eye admins should be added locally through third-eye-print-co/admins or ADMIN_PUBS.
